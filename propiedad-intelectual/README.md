# Propiedad Intelectual — PI e industrial

Este módulo cubre la protección de marcas, patentes, derechos de autor, secretos empresariales y software de código abierto. Asiste en búsquedas de anterioridades, redacción de requerimientos por infracción, análisis de licencias OSS, cláusulas de PI para contratos y gestión de portfolios de activos intangibles. Pensado para despachos de PI, departamentos legales de empresas tecnológicas y asesores de innovación.

## Skills disponibles

| Skill | Comando | Descripción |
|-------|---------|-------------|
| Búsqueda de marca | `/busqueda-marca` | Realiza un análisis preliminar de disponibilidad y riesgo de conflicto marcario |
| Burofax | `/burofax` | Genera borradores de requerimientos por infracción de derechos de PI |
| OSS | `/oss` | Analiza licencias de software de código abierto y detecta incompatibilidades |
| Cláusulas | `/clausulas` | Redacta cláusulas de PI para contratos (cesión, licencia, confidencialidad) |
| Portfolio | `/portfolio` | Gestiona y visualiza el estado de registros de marcas, patentes y dominios |

## Primeros pasos

1. Ejecuta la entrevista inicial del proyecto para configurar tu perfil y preferencias.
2. Prueba `/busqueda-marca` con un nombre comercial para evaluar su disponibilidad registral.
3. Usa `/oss` para auditar las dependencias de un proyecto de software y detectar licencias copyleft.
4. Ejecuta `/portfolio` para tener una vista general de todos los activos de PI registrados.

## Casos de uso típicos

- Búsqueda de anterioridades antes de solicitar el registro de una marca ante la OEPM.
- Redacción de un burofax por uso no autorizado de una marca registrada.
- Auditoría de licencias OSS en un producto de software antes de su comercialización.
- Preparación de cláusulas de cesión de PI en un contrato de desarrollo a medida.

## Legislación de referencia

- **Ley de Marcas** — Ley 17/2001, de 7 de diciembre, de Marcas
- **Ley de Patentes** — Ley 24/2015, de 24 de julio, de Patentes
- **LPI** — Real Decreto Legislativo 1/1996, Ley de Propiedad Intelectual
- **Ley de Secretos Empresariales** — Ley 1/2019, de 20 de febrero

## Lo que este plugin no hace

- **No redacta reivindicaciones de patente**: la redacción técnica de solicitudes de patente requiere un agente de patentes cualificado.
- **No elabora informes periciales**: no genera dictámenes técnicos válidos como prueba procesal.
- **No presenta solicitudes ante la OEPM ni la EUIPO**: no tramita registros de marcas ni patentes.
- **No sustituye el criterio profesional** del abogado: las sugerencias deben ser revisadas por un letrado colegiado.
